

# 使用说明

文档主要用于编写UIBE《非寿险精算学》的教学资料，教学资料托管到[静态网站](https://lizhengxiao.github.io/Non-life-Insurance-Actuarial-Science/)上，工作由[github](https://www.baidu.com/link?url=7yCbMudQ6Cs1wgDPGAsiYx3AVvkzJbZrO1d4Sgk_QGfVKSlEpbLfgWSCPpy-xYJe3R1GoyAhl4T3h1bwC16LV2Z7d12mHWURwa9_xvrVuQy&wd=&eqid=cc408585001446e000000006625fc55d)和rmarkdown完成。

github项目网址：https://github.com/lizhengxiao/Non-life-Insurance-Actuarial-Science

编写文员需要首先按照下述步骤进行操作：


1. 下载 github desktop 并且注册，主要用于项目托管，版本控制，本地编写上传等工作，可见[相关说明](https://www.jianshu.com/p/06a960d991aa).
2. 熟悉 github desktop 的[基本操作](https://www.jianshu.com/p/8c69d1021d98?msclkid=71161c90c08211ecab1b6bdfd8c4ae7b),熟悉fork, branch, pull Request, push 等用法进行多人项目管理，加入项目团队。
3. 安装 R 软件中的 bookdown 包，主要用于教材资料编写整理。 bookdown 的资料可以参考：
	- https://www.jianshu.com/p/ecb3276e151f
	- https://bookdown.org/
	- https://bookdown.org/yihui/bookdown/
4. 加入项目，托管本地，修改.rmd文件进行教案整理并上传，上传需要写上修改描述。
